Contentful stores all your content — photos, publications, talks, bio text — so it persists across devices and sessions. Without it, changes are lost on refresh.

Once connected, all sections of this admin panel save directly to Contentful and appear live on the site immediately.

Connect Contentful

One-time setup · Free tier is enough

  1. Go to contentful.com and create a free account
  2. Create a new Space (any name, e.g. "pignocchi-photo")
  3. In Settings → API keys → "Add API key" — copy the Space ID and Content Delivery API token
  4. In Settings → API keys → "Content management tokens" — generate a Management token
  5. Paste all three below and click Setup
